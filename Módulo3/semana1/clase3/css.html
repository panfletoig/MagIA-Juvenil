Los diseñadores Web utilizan listas de viñetas para
algo más que agregar viñetas y texto. La etiqueta HTML
utilizado para crear listas de bala es muy útil para
la creación de menús o la organización de imágenes para
un JavaScript para cargar en una presentación de
diapositivas. Aprender a usar el código de hojas de estilo
en cascada junto con listas de viñetas - llamados
"listas desordenadas" en el argot HTML - se abrirá varios
emocionantes posibilidades Web-diseño.
